## Creating This Web Site

**RolloTheGreat.com** is 100% built on Open Source software, and is Open Source itself. 

Rollo's dad built it using the following tools:

* VirtualBox
* Linux O/S (Ubuntu)
* Ruby, Gems & Jekyll
* HTML
* Markdown
* Hydejack Theme
* Github
* Some Python plugins or scripts coming soon
